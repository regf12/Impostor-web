# **App Name**: ImpostorSpot

## Core Features:

- Game Lobby: Create and join game lobbies with custom settings.
- Role Assignment: Randomly assign roles (Impostor/Crewmate) to players in the game.
- Task Management: List of tasks that crewmates need to accomplish.
- Reporting System: Enable players to report suspicious activity or identify potential Impostors.
- Emergency Meetings: Allow players to call emergency meetings for discussions and voting.
- Voting Mechanism: Implement a voting system for players to eliminate suspected Impostors.
- Single Device Gameplay: Enable playing on a single device by selecting the number of players and impostors. Upon starting the game, a button reveals each player's role and a hint to the impostors (if enabled). A button advances to the next player. The game must be fully in Spanish, and generate words and hints in Spanish.
- Game Timer: Display a 15-minute timer for each game session.
- Voting and Outcome: Implement a voting process to select the Impostor and determine if the Impostor was discovered or not.

## Style Guidelines:

- Primary color: Deep Red (#B71C1C) for the danger and tension of the game.
- Background color: Dark Gray (#303030) for a suspenseful atmosphere.
- Accent color: Electric Blue (#2962FF) for highlights and interactive elements.
- Body and headline font: 'Inter', a sans-serif font providing a modern, readable style, well-suited to both headers and blocks of text.
- Use icons from a set that's simple, bold and universally understandable. For example, use icons from the 'Material Icons' set.
- The layout should emphasize clear communication. The layout should be simple and intuitive, emphasizing important UI elements.
- Implement subtle animations when a user performs actions (e.g. voting, task completion) to acknowledge that their actions have been registered by the system.
